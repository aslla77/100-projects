## 0. 요약 (먼저 이것만 읽어도 됨)

**마이리얼트립이 뭐 하는 회사인가:** 비행기표·호텔·현지 투어·렌터카·여행 패키지를 한 개의 앱에서 검색하고 예약하게 해주는 회사다. 여러 여행 상품을 한곳에 모아 파는 "온라인 여행사(OTA)"이자, 현지 가이드 같은 공급자와 여행자를 연결하는 "플랫폼"이다 [11][16][18].

**이 보고서로 알 수 있는 것:** 회사가 무엇을 파는지, 누가 고객인지, 매출·영업이익·순이익의 **연도별 추세와 방향**, 2025년부터 왜 "AI 전환"을 회사 정체성으로 내세웠는지, 그리고 그 전환이 가리키는 **실제 기술/사업 문제 후보**.

**이 보고서로 알 수 없는 것(근거 부족):** ① FY2024·FY2025의 **당기순이익·판매관리비·지급수수료 같은 세부 비용 라인 수치**(DART 감사보고서 본문 라인아이템은 이번 조사에서 직접 추출하지 못함 — 아래 4장·근거 부족 모음 참조), ② 자회사·M&A 완료 내역, ③ "AI 챔피언"이 자동화했다는 구체적 반복업무 목록. 모두 정직하게 "근거 부족"으로 남겼다.

## 4. 재무 신호 (Financial Signals) — 사실만, 출처·회계연도 명시

> 이 장은 research-brief의 "영업이익 출처 충돌"을 해소하려고 DART(전자공시) 우선으로 보강한 부분이다. **결론부터:** 매출과 영업이익의 *방향*은 DART 감사보고서 존재 + 다수 매체로 확정되지만, **순이익·판관비·지급수수료 같은 세부 라인 수치는 DART 본문에서 직접 확정하지 못해 "근거 부족"으로 남긴다.** 아는 척하지 않는다.

### 4-2. 핵심 사실 (확정)

- `[사실]` FY2024: 매출 892억 원으로 전년 대비 47.3% 증가, 영업이익 약 1.3억 원으로 **창사 이래 첫 연간 흑자**. 다수 매체가 일관되게 보도했고 [4][33](B), 회사 뉴스룸도 발표했다. 해당 회계연도 **감사보고서가 DART에 등록돼 있다(접수번호 20250403003706, 2025-04-03 제출)** [27](A) — 즉 1차 공시 원본이 존재한다.
- `[사실]` FY2025: 매출 1,120억 원(+26%), 총거래액 약 2.3조 원, 당기순이익 약 +3.1억 원(흑자전환). 여러 매체가 회사 발표를 인용 보도 [19][21][31][32](B).
- `[사실 — 충돌 해소]` **FY2025 영업이익은 약 -20억 원 적자(적자전환)**다. research-brief가 "충돌, 재확인 필요"로 남겼던 지점이다. 여행신문 보도가 명시적으로 설명한다: *"AI 기술 내재화와 소수 정예 인재 영입 등 미래 성장을 위한 선제적 투자에 집중하며 영업이익 기준으로는 약 20억 원 손실이 발생하며 적자 전환"* [20](B). 즉 다른 매체들이 강조한 "당기순이익 흑자"와 모순이 아니라, **영업이익=적자 / 당기순이익=흑자**가 동시에 사실이다(둘은 다른 지표 — 용어 설명 참조).

### 4-3. 재무 신호가 가리키는 것 (사실 기반 해석, 가설은 별도 표기)

- `[사실]` FY2024(영업 흑자 1.3억) → FY2025(매출 +26%인데 영업이익 -20억 적자전환)로, **"매출은 늘었는데 영업이익은 마이너스로 꺾인"** 전형적 신호가 나타난다.
- `[사실]` 회사가 밝힌 영업적자 원인: AI 내재화·소수 정예 인재 영입 등 **선제 투자** [20](B). 동시에 광고 등 플랫폼 기타매출이 전년比 +390% 급증해 새 수익원으로 부상 [19][21](B).
- `[가설]` 영업적자(-20억)인데 당기순이익(+3.1억)이 흑자인 것은 영업외수익(예: 투자·금융 수익, 환입 등)이 영업손실을 상쇄했을 *가능성*이 있다 — 단, **구체 항목은 DART 본문 미확인으로 근거 부족**. "추정"이며 사실로 쓰지 않는다.

### 4-4. DART로 확정한 범위 / 못 한 범위 (정직한 경계)

- ✅ 확정: FY2024 감사보고서가 DART에 실재(rcpNo 20250403003706)[27], FY2021~2023 감사보고서도 실재 [28] 외. 매출·영업이익의 *값과 방향*은 1차 공시 존재 + 다수 B급 보도로 교차 확인.
- ❌ 미확정(근거 부족): DART 공시 뷰어가 프레임 구조라 이번 조사에서 **재무제표 본문 라인아이템(당기순이익 정확치, 판매관리비, 지급수수료, 재고/충당부채, 현금흐름)**을 직접 추출하지 못했다. FY2024 당기순이익은 보도에도 수치가 없다. → **세부 비용·순이익 구조는 "근거 부족", DART 감사보고서 본문 정독으로 재확인 필요.**
- ❌ FY2025 감사보고서: 2026-06-28 현재 DART 검색에서 확인되지 않음(미접수 또는 미인덱스) → **FY2025 수치는 전적으로 B급 보도(회사 발표 인용)**다. A급 1차 확정은 추후 FY2025 감사보고서 공시 시 재확인 필요.

## 6. 문제 신호 → 이슈 가설

> **[사실]**(공개 출처로 입증된 문제)과 **[가설]**(추정·예측)을 엄격히 분리한다.
> ⚠️ 유튜브 'AX 인재전쟁' 2화에서 예고된 **럭키글라이드형 가격탐색 본선 문제는 그대로 쓰지 않는다.** 아래는 독립 공개 출처로 입증한 것과, 본선 회피용 *인접 변형*만 담았다.

**[확인된 사실 — 회사/도메인이 실제로 안고 있는 문제]**
- 항공권 가격이 도시·일정마다 크게 변동해 사용자가 반복 검색해야 한다 → 회사가 럭키글라이드로 직접 정의·공략 [6][9](B).
- 이종 공급사(항공 GDS/OTA, 숙소, 수많은 가이드 파트너)의 상품을 단일 상품DB로 정규화·관리해야 한다 → 백엔드 JD에 명시 [11][12](A).
- "9명이 하던 일을 3명이" 식 RPE 중심 운영을 위해 반복업무를 AI로 자동화해야 한다 → 대표 공식 글에 명시 [15](A).

**[이슈 예측 = 가설]**
- `[가설]` FY2025 영업적자(-20억)가 지속되면 RPE·자동화 압박이 더 커질 *가능성*. (재무 신호 기반 추정 — 사실 아님)

### 문제 후보 (Codex 적합성 3축: 입력데이터 공개가용성 / 문제경계 명확성 / 플러그인 표현가능성, 각 상·중·하)

**P1. 가격 기반 "역방향" 여행 탐색 — 단, 숙박·액티비티로의 인접 변형** `[사실]`
- 한 줄: "여행지 먼저"가 아니라 "예산(가격) 먼저" 정하고 그 안에서 목적지·날짜·상품을 찾는 탐색. 본선 예고된 항공권판(럭키글라이드)은 제외하고, **회사가 "확대 예정"이라 공개 선언한 숙박·액티비티 가격탐색**으로 변형.
- 출처: 럭키글라이드 보도 + "숙박·액티비티로 확대 예정" 언급 [6][9](B); 본선 회피 근거 = 회사 스스로 인접 영역을 공개 [7](B).
- Codex 적합성: 입력데이터 **중**(항공 캘린더 API는 비공개지만 숙박·액티비티 가격은 공개 OTA/더미로 대체 가능) / 문제경계 **상**(가격→후보 추천으로 좁음) / 플러그인 표현 **상**(가격 통계·대안추천 스킬). → **종합 적합도 상**.

**P2. 이종 상품 피드 정규화 / 상품DB 스키마 린터** `[사실]`
- 한 줄: 서로 다른 포맷(JSON/CSV)으로 들어오는 항공·숙소·투어 상품 피드를 표준 스키마로 매핑·검증하고, 누락·중복·이상치를 잡아내는 도구.
- 출처: 백엔드 JD "OTA/GDS 연동", "효율적 상품탐색·상품정보 관리를 위한 상품DB 설계", "대규모 서비스 설계·운영" [11][12](A); 기술블로그 [14](A).
- Codex 적합성: 입력데이터 **상**(공개 OTA 스펙·더미 피드로 충분) / 문제경계 **상**(스키마 매핑·검증으로 명확) / 플러그인 표현 **상**(린팅/정규화 스킬+스크립트). → **종합 적합도 상**. (본선 문제와 무관 → 차별화에 유리)

**P3. 환불·취소 약관 해석 계산기** `[사실(문제 존재) + C단서(불만 사례)]`
- 한 줄: 상품(항공/숙소/투어)별로 제각각인 환불 규정을 입력받아 "이 예약을 지금 취소하면 수수료/환불액이 얼마"인지 계산·설명.
- 출처: 환불 규정 복잡성 자체 = 공식 고객지원 안내(A) [22]; 불만·분쟁 *사례*는 traveli.net·brunch(C단서) [23][24] — **문제의 '존재'는 A(공식 약관 복잡성)로만 입증**, C는 방향 단서로만.
- Codex 적합성: 입력데이터 **중**(약관 텍스트는 공개·더미화 용이) / 문제경계 **상**(약관→금액 계산으로 좁음) / 플러그인 표현 **상**(규칙 해석 스킬). → **종합 적합도 중~상**.

**P4. 반복 운영업무 자동화 보조 도구 (RPE 향상)** `[사실]`
- 한 줄: 가격·리뷰·운영 데이터 분석, 반복 문서/CS 응대 초안 생성 등 현업 반복 작업을 자동화하는 사내 도구형 플러그인.
- 출처: 대표 공식 글(소수정예·RPE·AI 챔피언) [15](A); 벤처스퀘어(약 30종 AI 도구) [16](B); 이투데이 [21](B).
- Codex 적합성: 입력데이터 **하~중**("어떤 반복업무"인지 구체 데이터가 비공개 — 더미로 대체하나 현실성 약함) / 문제경계 **중**(범위가 넓어질 위험) / 플러그인 표현 **상**. → **종합 적합도 중**. (AX 인재전쟁 취지엔 부합하나 경계가 넓음)

## 용어 설명 (초심자용 한 줄 정의)

- **OTA (Online Travel Agency / 온라인 여행사):** 항공·숙소·투어 등 여행 상품을 인터넷에서 모아 파는 회사. 예: 마이리얼트립, 아고다, 부킹닷컴. (출처: 업계 일반 통용어 — [17] 참고)
- **GDS (Global Distribution System / 글로벌 항공 예약망):** 항공사·호텔의 좌석·요금 정보를 여행사에 실시간 공급하는 글로벌 예약 전산망(예: Amadeus). OTA는 GDS와 연동해 항공권을 판다. (JD에 "OTA/GDS 연동" 명시 [11][12])
- **MAU (Monthly Active Users / 월간 활성 이용자):** 한 달 동안 앱을 실제로 쓴 이용자 수. 마이리얼트립 500만+ [19].
- **총거래액 (GMV, Gross Merchandise Value):** 플랫폼에서 거래된 상품 금액의 총합. 회사 매출과 다르다(매출은 그중 수수료 등만). 마이리얼트립 FY2025 약 2.3조 [19].
- **Take rate (수수료율):** 총거래액 중 회사가 매출로 가져가는 비율 = 매출 ÷ 총거래액.
- **RPE (Revenue Per Employee / 1인당 매출):** 매출 ÷ 직원 수. "소수 정예" 효율의 지표로 회사가 강조 [15].
- **영업이익 vs 당기순이익:** **영업이익** = 본업(매출-매출원가-판관비)에서 번 돈. **당기순이익** = 영업이익에 이자·투자·세금 등 영업 외 손익까지 다 반영한 최종 이익. **둘은 부호가 다를 수 있다**(FY2025: 영업 -20억, 순이익 +3.1억). (회계 일반 정의)
- **DART (전자공시시스템, Data Analysis, Retrieval and Transfer System):** 금융감독원이 운영하는 기업 공시 사이트. 외부감사 대상 회사의 감사보고서·재무제표 원본을 공개한다. 가장 신뢰도 높은 1차 출처(A). https://dart.fss.or.kr [27]
- **감사보고서:** 외부 회계법인이 감사한 회사의 1년치 재무제표 + 의견서. DART에 매년 등록된다 [27][28].
- **FIT (Free Independent Traveler / 자유개별여행):** 패키지투어가 아니라 개인이 일정·상품을 직접 짜는 여행자. 마이리얼트립의 주 B2C 고객 [17].
- **시리즈F:** 스타트업 투자 단계 명칭(A→B→…→F). 뒤로 갈수록 후기·대형 라운드. 마이리얼트립은 2024년 시리즈F 756억 [2].

## 근거 부족 모음 (정직하게 남기는 미확인 항목)

1. **FY2024·FY2025 당기순이익 정확치, 판매관리비·지급수수료·재고·현금흐름** — DART 감사보고서가 실재하나(rcpNo 20250403003706 [27]) 본 조사에서 공시 뷰어 프레임 구조 때문에 재무제표 라인아이템 본문을 직접 추출하지 못함. FY2024 순이익은 보도에도 수치 없음. → 감사보고서 본문 정독으로 재확인 필요.
2. **FY2025의 A급(감사보고서) 확정 수치** — 2026-06-28 현재 DART에서 FY2025 감사보고서 미확인(미접수/미인덱스). FY2025 매출·영업이익·순이익은 모두 B급 보도(회사 발표 인용)뿐. → 추후 공시 시 A급 재확인.
3. **자회사·M&A 완료 내역** — "글로벌 M&A 추진" 전략 보도만 있고 구체 명단 공개 불가 [18]. → DART 종속회사 주석·법인 등기 확인 필요.
4. **"AI 챔피언"이 자동화한 구체 반복업무 목록** — 대표 글은 추상적 [15]. → 추가 기술블로그·JD 정독 필요.
5. **세그먼트별 매출(B2C/B2B/광고) 분해** — 비상장사라 공개 한계. FY2025 기타매출 +390%의 절대액 미상.
6. **럭키글라이드의 "항공 캘린더 API" 출처/스펙** — 자체 GDS 연동인지 외부 제공인지 공개 확인 불가.
