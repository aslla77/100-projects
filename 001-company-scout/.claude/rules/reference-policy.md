# Rule: 출처 정책

- 공식 문서 우선. 표준 문서(RFC/W3C/ISO) 우선.
- 보안은 OWASP·벤더 공식 문서·CVE·NVD 등 신뢰 가능한 출처 우선.
- 커뮤니티 / 블로그 / Reddit / 개인 글 / 유튜브 / 위키는 단서로만 사용.
- 출처가 부족하면 부족하다고 말한다.
- 사실 / 가정 / 의견 / 추천을 분리한다.
- 모든 분석 문서 마지막에 References 섹션을 추가한다.
- 사실 주장에는 본문에 `[n]` 번호를 달고 References에 매핑한다.

## 출처 등급

- B: 평판 있는 기술 매체·공식 블로그
- C: 커뮤니티·개인·유튜브·위키 (단서 전용)
