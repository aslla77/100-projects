# ADR 0002: 공개 출처 한정 + A/B/C 등급 + 근거부족 분리 정책

## 상태
제안됨 (날짜: 2026-06-28)

## 배경 (Context)
- 예선 채점은 **공개·검증 가능 자료만** 근거로 인정한다. 내부 비공개·확인 불가 사적경험·출처 없는 숫자는 불인정 [1]. 이는 프로젝트 제0지침과도 일치.
- 일반 LLM 산출물은 출처 없는 환각 위험 → 채점 탈락 직결(R2).

## 결정 (Decision)
- SKILL.md가 **모든 사실 주장에 공개 URL 출처 + 등급(A=공식/표준, B=평판 매체, C=커뮤니티·유튜브·앱리뷰 단서)** 을 강제한다.
- 출처를 댈 수 없는 항목은 **"근거 부족" 섹션으로 분리**하고 사실로 단정하지 않는다.
- 출처 충돌 시 양쪽 병기 + "재확인 필요" 표기.
- C등급(유튜브·커뮤니티)은 **단서로만**, 사실 단정에 쓰지 않는다. 본선 문제(유튜브 예고)는 독립 공개 출처로 별도 입증된 경우만 채택.

## 검토한 선택지 (Options considered)
| 선택지 | 장점 | 단점 | 출처 |
|---|---|---|---|
| A (채택) 출처 강제 + 등급 + 근거부족 분리 | 예선 규칙·제0지침 일치, 환각 차단 | 산출량 적어질 수 있음(정직) | [1] |
| B 출처 권장(강제 아님) | 빠름·풍부 | 출처 없는 주장 혼입 → 탈락 위험 | [1] |

## 결과와 영향 (Consequences)
- 긍정: 제출물이 채점 기준에 안전. dossier가 그대로 기획 근거가 됨.
- 트레이드오프: 근거 없는 화려한 주장은 못 씀(의도된 제약).

## 되돌리거나 재검토할 조건
- 없음. 이 정책은 예선 채점과 제0지침의 직접 요구라 유지.
