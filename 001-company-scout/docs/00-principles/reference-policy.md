# 제0지침 & 출처 정책

이 문서는 이 프로젝트의 모든 분석·제안·문서에 우선하는 기본 운영 철학이다.

## 제0지침

```
1. 모르면 모른다고 말한다.
2. 아는 척하지 않는다.
3. 섣부른 추론을 금지한다.
4. 추론이 필요하면 먼저 질문하거나 명확히 "가정"으로 표시한다.
5. 설명에는 항상 레퍼런스를 추가해 깊이를 더한다.
6. 모든 분석·제안·문서의 마지막에 레퍼런스를 첨부한다.
7. 커뮤니티·블로그·유튜브·위키는 단서로만 사용한다.
8. 중요한 결정에는 공식 문서·표준·제품 문서·논문·법령·보안 가이드 등 공신력 있는 정보를 우선한다.
9. 출처가 부족하면 "현재 확인 가능한 근거가 부족하다"고 말한다.
10. 사실·가정·의견·추천을 구분해서 작성한다.
```

## 출처 등급

| 등급 | 유형 | 사용 |
|---|---|---|
| A (1차/공식) | 공식 문서, 표준(RFC/W3C/ISO), 벤더 문서, 논문, 법령, OWASP/CVE/NVD | 중요 결정의 근거로 사용 |
| B (검증된 2차) | 평판 있는 기술 매체, 공식 블로그 | 보조 근거 |
| C (단서) | 커뮤니티, 개인 블로그, 유튜브, 위키 | 단서로만, 단정 금지 |
